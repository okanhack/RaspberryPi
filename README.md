# raspberryPi
RaspberryPi program sample.
Site：okan89-blog.hatenablog.com/


## プログラムの内容
### led1.py
LEDを点滅させるサンプルです。

### distanceSensor.py
超音波距離センサーを使い、距離を出力します。

### switch.py
スイッチに連動してLEDを光らせるサンプルです。

### switch2.py
switch.pyをイベント駆動型にしました。

### switch3.py
LEDを3つ繋げて、スイッチを押した回数により光るLEDを増やすプログラムです。
